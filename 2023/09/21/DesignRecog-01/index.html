设计与认知

  • 信息设计中的格式塔原则
    • 一套人类感知的原则
    • 描述了人类在感知物体时如何
      • 组织彼此相似的元素 / 检测模式 / 澄清复杂的图像
    • Base Rules
      • 看事物或文字时，看到事物最基本的形状。
      • 在任何文本或元素中追踪线条 / 曲线。
      • 大脑试图 感知并填补 环境中 不存在 的细节。
    • Main Rules
      • 连续性 / 当眼睛开始跟随任何东西，视线都会继续朝该方向移动，直到遇到另一个物品。
      • 相似性 / 大脑将任何两个具有相似外在形状的物品解释为属于同一个实体。
      • 接近性 / 当两个或多个元素接近时，这些元素的位置描绘了不同元素之间的关系。
      • 共同区域 / 大脑将几个物品在同一封闭区域内的存在解释为这些物体属于一起的标志。
      • 对称法，Pragnanz / 一种基本的、和谐的规范，一种秩序和正确的感觉。
      • 主体性 / 人眼将一个物品从其周围环境中区分出来。
      • 闭合性 / 人脑偏爱整个形状或图像，填补元素之间的空白以看到一个完整的图像，产生一个整体。
